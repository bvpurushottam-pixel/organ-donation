# Centralized Organ Donation & Tracking Platform

A simplified prototype of a real-time organ donation and tracking platform that connects donors, hospitals, and patients. Built for academic demonstration to simulate critical workflows such as donor registration, hospital search, organ tracking, and live status updates.
